## 1. 求解思路

教材给出的求解逻辑是**先齐次、后非齐次**：

- 第一步：掌握齐次方程 $$P(x) \cdot \frac{d^2 y}{dx^2} + Q(x) \cdot \frac{dy}{dx} + R(x) \cdot y = 0$$ 的解法，找到它的通解；
- 第二步：利用齐次方程的通解，通过 “常数变易法”“待定系数法” 等方法，求解非齐次方程 $$P(x) \cdot \frac{d^2 y}{dx^2} + Q(x) \cdot \frac{dy}{dx} + R(x) \cdot y = f(x) $$。
研究的非齐次线性 ODE 形式：$P(x) \cdot \frac{d^2 y}{dx^2} + Q(x) \cdot \frac{dy}{dx} + R(x) \cdot y = f(x)$
- **非齐次项**：$f(x) \neq 0$（若 $f(x)=0$ 就是齐次方程 ）；
- **$y_H$**：对应**齐次方程的通解**（互补解），即解 $P(x) \cdot \frac{d^2 y}{dx^2} + Q(x) \cdot \frac{dy}{dx} + R(x) \cdot y = 0$ 得到的 $y_H$；

- **$y_P$**：非齐次方程的**一个特解**（随便找一个满足原方程的解即可 ）。
**核心结论**：非齐次方程的**通解**是齐次通解与非齐次特解的和，即：$y = y_H + y_P$

---
## 核心解法
这是**二阶常系数线性齐次微分方程**的核心解法，按 3 步理解：

##### 1. 方程形式

研究的微分方程是：$$\boldsymbol{a \cdot \frac{d^2 y}{dx^2} + b \cdot \frac{dy}{dx} + c \cdot y = 0}$$ 其中 $\boldsymbol{a,b,c}$ 是**常数**（所以叫 “常系数” ），目标是找它的通解 $y_H(x)$ 。

##### 2. 通解的构造逻辑

根据线性微分方程解的结构，只要找到 **2 个线性无关的解 $y_1, y_2$**，通解就是：$$\boldsymbol{y_H(x) = A \cdot y_1(x) + B \cdot y_2(x)}$$ （$A,B$ 为任意常数 ）
#### 情况 1：$\boldsymbol{\Delta > 0}$（两个不同实根 $k_1, k_2$ ）

根的形式：$\boldsymbol{k_{1,2} = \frac{ -b \pm \sqrt{\Delta} }{2a}}$

  

此时，微分方程的 2 个线性无关解是：$\boldsymbol{y_1(x) = e^{k_1 x}, \quad y_2(x) = e^{k_2 x}}$

  

通解就是：$\boldsymbol{y_H(x) = A e^{k_1 x} + B e^{k_2 x}}$

#### 情况 2：$\boldsymbol{\Delta = 0}$（一个二重实根 k ）

根的形式：$\boldsymbol{k = -\frac{b}{2a}}$

  

此时，微分方程的 2 个线性无关解是：$\boldsymbol{y_1(x) = e^{k x}, \quad y_2(x) = x \cdot e^{k x}}$

  

通解就是：$\boldsymbol{y_H(x) = (A + B x) e^{k x}}$

#### 情况 3：$\boldsymbol{\Delta < 0}$（一对共轭复根 $k = \alpha \pm i\beta$ ）

根的形式：$\boldsymbol{k = \frac{ -b \pm i\sqrt{-\Delta} }{2a}}$ （可拆成 $\alpha + i\beta$ 和 $\alpha - i\beta$ ，其中 **$$\alpha = -\frac{b}{2a}$$$$ \beta = \frac{\sqrt{-\Delta}}{2a}$$** ）

利用欧拉公式 $e^{i\theta} = \cos\theta + i\sin\theta$，微分方程的 2 个线性无关解可表示为：$\boldsymbol{y_1(x) = e^{\alpha x} \cos(\beta x), \quad y_2(x) = e^{\alpha x} \sin(\beta x)}$

通解就是：$\boldsymbol{y_H(x) = e^{\alpha x} \left( A \cos(\beta x) + B \sin(\beta x) \right)}$

## 特解 $y_P$ 的求解思路
### 一、核心方法分类：4 种常用策略

针对常系数二阶非齐次线性 ODE $a y'' + b y' + c y = f(x)$，特解 $y_P$ 的求解方法，本质是 “**根据非齐次项 $f(x)$ 的形式，匹配特解假设形式**”，再结合 “待定系数法” 或 “常数变易法” 确定系数。常用方法有：

| 方法名称        | 适用场景                       | 核心逻辑                                                  |
| ----------- | -------------------------- | ----------------------------------------------------- |
| **待定系数法**   | $f(x)$ 为多项式、指数、三角、它们的乘积  | 假设特解为与 $f(x)$ 同类型的函数（带待定系数），代入方程解系数                 |
| **常数变易法**   | 通用方法（对 $f(x)$ 无限制）       | 将齐次通解的 “常数” 换成 “函数”，代入非齐次方程解出这两个函数                    |
| **叠加原理**    | $f(x) = f_1(x) + f_2(x)$ | 分别求 $f_1,f_2$ 对应的特解 $y_{P1}, y_{P2}$，再相加得 $y_P$ |
| **拉普拉斯变换法** | 适合初值问题（有初始条件）              | 对微分方程做拉普拉斯变换，转化为代数方程求解，再逆变换得特解（进阶方法）                  |

下面重点讲最常用的 **待定系数法** 和 **常数变易法**（拉普拉斯变换法后续单独展开）。

### 二、方法 1：待定系数法（最常用！）

#### 核心思想

若非齐次项 $f(x)$ 是 **多项式、指数函数、三角函数、或它们的乘积**（如 $x e^{kx}, x^2 \sin \beta x$ 等），则特解 $y_P$ 的形式可 **“模仿” $f(x)$ 的结构 **，带待定系数，代入方程后解出系数即可。

#### 分情况讨论（根据 $f(x)$ 形式）

表格给出了 **非齐次项 $f(x)$ 形式** 与 **特解 $y_p(x)$ 假设形式** 的对应关系，本质是 “待定系数法” 的 “模板匹配” 逻辑：

|$f(x)$ 的形式|尝试的特解 $y_p(x)$ 形式|核心思路|
|---|---|---|
|次数为 r 的多项式|次数为 r 的多项式|多项式的导数仍为低次多项式，形式可匹配|
|$A \cdot e^{kx}$（A 为常数）|$\alpha \cdot e^{kx}$（$\alpha$ 为常数）|指数函数求导后形式不变（$(e^{kx})' = k e^{kx}$）|
|$A\cos(kx) + B\sin(kx)$（$A,B$ 为常数）|$\alpha\cos(kx) + \beta\sin(kx)$（$\alpha,\beta$ 为常数）|三角函数求导后仍为同频率的正 / 余弦组合|


设齐次方程的特征方程为 $a k^2 + b k + c = 0$，根为 $k_1, k_2$（可相同、复数），分以下 4 类情况：

##### 情况 1：$f(x) = P_n(x)$（$P_n(x)$ 是 n 次多项式 ）

- **假设特解形式**：
    
    - 若 $c \neq 0$（即 $k=0$ 不是特征根 ）：设 $y_P = Q_n(x)$（$Q_n(x)$ 是 n 次待定多项式，如 $Q_n = A_0 + A_1 x + \dots + A_n x^n$ ）
    - 若 $c = 0$ 但 $b \neq 0$（即 $k=0$ 是单特征根 ）：设 $y_P = x \cdot Q_n(x)$
    - 若 $c = 0$ 且 $b = 0$（即 $k=0$ 是二重特征根 ）：设 $y_P = x^2 \cdot Q_n(x)$
- **示例**：解方程 $y'' + y' = x^2$（$f(x) = x^2$ 是二次多项式 ）
    
    - 齐次方程 $y'' + y' = 0$，特征方程 $k^2 + k = 0$，根 $k=0, k=-1$ → $k=0$ 是单特征根（因 $c=0, b=1 \neq 0$ ）
    - 设特解 $y_P = x(A x^2 + B x + C) = A x^3 + B x^2 + C x$
    - 求导：$y_P' = 3A x^2 + 2B x + C$，$y_P'' = 6A x + 2B$
    - 代入原方程：$(6A x + 2B) + (3A x^2 + 2B x + C) = x^2$
    - 整理：$3A x^2 + (6A + 2B) x + (2B + C) = x^2$
    - 比较系数：$3A = 1$ → $A = 1/3$$6A + 2B = 0$ → $6*(1/3) + 2B = 0$ → $B = -1$$2B + C = 0$ → $2*(-1) + C = 0$ → $C = 2$
    - 特解：$y_P = \frac{1}{3}x^3 - x^2 + 2x$

##### 情况 2：$f(x) = P_n(x) e^{\alpha x}$（多项式 × 指数函数 ）

- **假设特解形式**：
    
    - 若 $\alpha$ 不是特征根（即 $a \alpha^2 + b \alpha + c \neq 0$ ）：设 $y_P = Q_n(x) e^{\alpha x}$（$Q_n(x)$ 是 n 次待定多项式 ）
    - 若 $\alpha$ 是单特征根（即 $a \alpha^2 + b \alpha + c = 0$ 但 $2a \alpha + b \neq 0$ ）：设 $y_P = x \cdot Q_n(x) e^{\alpha x}$
    - 若 $\alpha$ 是二重特征根（即 $a \alpha^2 + b \alpha + c = 0$ 且 $2a \alpha + b = 0$ ）：设 $y_P = x^2 \cdot Q_n(x) e^{\alpha x}$
- **示例**：解方程 $y'' - 2y' + y = x e^x$（$f(x) = x e^x$，$\alpha = 1$ ）
    
    - 齐次方程特征方程 $k^2 - 2k + 1 = 0$，根 $k=1$（二重根，因 $(k-1)^2 = 0$ ）
    - 因 $\alpha = 1$ 是二重特征根，设特解 $y_P = x^2 (A x + B) e^x = (A x^3 + B x^2) e^x$
    - 求导（用乘积法则）：$y_P' = (3A x^2 + 2B x) e^x + (A x^3 + B x^2) e^x = [A x^3 + (3A + B) x^2 + 2B x] e^x$$y_P'' = [3A x^2 + 2(3A + B) x + 2B] e^x + [A x^3 + (3A + B) x^2 + 2B x] e^x = [A x^3 + (6A + B) x^2 + (6A + 4B) x + 2B] e^x$
    - 代入原方程 $y'' - 2y' + y = x e^x$，约去 $e^x$ 后：$[A x^3 + (6A + B) x^2 + (6A + 4B) x + 2B] - 2[A x^3 + (3A + B) x^2 + 2B x] + [A x^3 + B x^2] = x$
    - 整理左边：$A x^3 + (6A + B) x^2 + (6A + 4B) x + 2B - 2A x^3 - (6A + 2B) x^2 - 4B x + A x^3 + B x^2 = 6A x + 2B$
    - 比较系数：$6A = 1$ → $A = 1/6$$2B = 0$ → $B = 0$
    - 特解：$y_P = \frac{1}{6} x^3 e^x$

##### 情况 3：$f(x) = P_n(x) e^{\alpha x} \cos\beta x$ 或 $P_n(x) e^{\alpha x} \sin\beta x$（多项式 × 指数 × 三角函数 ）

- **假设特解形式**： 利用**欧拉公式**，可将三角函数转化为复指数 $e^{(\alpha \pm i\beta) x}$，因此特解形式统一设为：
    
    - 若 $\alpha \pm i\beta$ 不是特征根：设 $y_P = e^{\alpha x} [Q_n(x) \cos\beta x + R_n(x) \sin\beta x]$（$Q_n, R_n$ 是 n 次待定多项式 ）
    - 若 $\alpha \pm i\beta$ 是特征根：设 $y_P = x e^{\alpha x} [Q_n(x) \cos\beta x + R_n(x) \sin\beta x]$
- **示例**：解方程 $y'' + y = \cos x$（$f(x) = \cos x$，即 $\alpha=0, \beta=1$，$P_n(x)=1$ ）
    
    - 齐次方程特征方程 $k^2 + 1 = 0$，根 $k = \pm i$ → $\alpha \pm i\beta = 0 \pm i \cdot 1$ 是特征根（单根）
    - 设特解 $y_P = x (A \cos x + B \sin x)$（因是单根，乘 x；$Q_n, R_n$ 是常数，因 $P_n(x)$ 是 0 次多项式 ）
    - 求导：$y_P' = (A \cos x + B \sin x) + x (-A \sin x + B \cos x)$$y_P'' = -2A \sin x + 2B \cos x + x (-A \cos x - B \sin x)$
    - 代入原方程 $y'' + y = \cos x$：$[-2A \sin x + 2B \cos x + x (-A \cos x - B \sin x)] + [x (A \cos x + B \sin x)] = \cos x$
    - 整理后：$-2A \sin x + 2B \cos x = \cos x$
    - 比较系数：$-2A = 0$ → $A = 0$$2B = 1$ → $B = 1/2$
    - 特解：$y_P = \frac{1}{2} x \sin x$

##### 情况 4：$f(x)$ 是多种形式的和（叠加原理）

若 $f(x) = f_1(x) + f_2(x)$（如 $f(x) = e^x + \sin x$ ），则分别求 $f_1, f_2$ 对应的特解 $y_{P1}, y_{P2}$，再令 $y_P = y_{P1} + y_{P2}$ 即可。

### 三、方法 2：常数变易法（通用方法）

#### 核心思想

不管 $f(x)$ 是什么形式，只要已知齐次方程的**两个线性无关解 $y_1, y_2$**，就可以将特解设为 $y_P = u(x) y_1 + v(x) y_2$（把齐次通解的 “常数” 换成 “函数” $u(x), v(x)$ ），代入非齐次方程，解出 $u(x), v(x)$。

#### 具体步骤（以二阶方程 $y'' + P(x) y' + Q(x) y = f(x)$ 为例）

1. **求齐次通解**：找到齐次方程的两个线性无关解 $y_1, y_2$，则齐次通解 $y_H = C_1 y_1 + C_2 y_2$。
    
2. **设特解形式**：令 $y_P = u(x) y_1 + v(x) y_2$，其中 $u(x), v(x)$ 是待定函数。
    
3. **列方程组**：为了确定 $u(x), v(x)$，需要两个方程。通常引入 [[ “一阶导数约束”]]： 令 $u' y_1 + v' y_2 = 0$（简化计算的技巧，避免出现二阶导数），则 $y_P' = u y_1' + v y_2'$。再求二阶导数 $y_P'' = u' y_1' + u y_1'' + v' y_2' + v y_2''$，代入原非齐次方程，结合齐次方程 $y_1'' + P y_1' + Q y_1 = 0$，$y_2'' + P y_2' + Q y_2 = 0$，可化简得到第二个方程：$u' y_1' + v' y_2' = f(x)$（或与 $a,b,c$ 相关的形式，取决于原方程系数 ）。
    
4. **解方程组**：联立 $\begin{cases} u' y_1 + v' y_2 = 0 \\ u' y_1' + v' y_2' = f(x) \end{cases}$ 用 **[[克莱姆法则]]** 或直接消元，解出 $u'(x), v'(x)$，再积分得到 $u(x), v(x)$。
5. **常见的方式**：
	$u_1=-\int\frac{y_2f(x)}{W(y_1,y_2)}dx$  
	$u_2=\int\frac{y_1f(x)}{W(y_1,y_2)}dx$
#### 示例：用常数变易法解 $y'' + y = \tan x$（$f(x) = \tan x$ 无法用待定系数法，因不是多项式 / 指数 / 三角的标准形式 ）

1. **齐次通解**：齐次方程 $y'' + y = 0$ 的解为 $y_1 = \cos x$，$y_2 = \sin x$，故 $y_H = C_1 \cos x + C_2 \sin x$。
2. **设特解**：$y_P = u(x) \cos x + v(x) \sin x$
3. **列方程组**：
    - 一阶导数约束：$u' \cos x + v' \sin x = 0$
    - 代入原方程求第二个方程：$y_P' = -u \sin x + u' \cos x + v \cos x + v' \sin x$$y_P'' = -u \cos x - u' \sin x - u' \sin x + u'' \cos x - v \sin x + v' \cos x + v' \cos x + v'' \sin x$ 代入 $y_P'' + y_P = \tan x$，利用 $u' \cos x + v' \sin x = 0$ 化简后，可得：$-u' \sin x + v' \cos x = \tan x$
    
    因此方程组为：$\begin{cases} u' \cos x + v' \sin x = 0 \\ -u' \sin x + v' \cos x = \tan x \end{cases}$
    
4. **解方程组**： 用克莱姆法则，系数矩阵行列式 $W = \begin{vmatrix} \cos x & \sin x \\ -\sin x & \cos x \end{vmatrix} = \cos^2 x + \sin^2 x = 1$$u' = \frac{\begin{vmatrix} 0 & \sin x \\ \tan x & \cos x \end{vmatrix}}{W} = -\sin x \tan x = -\frac{\sin^2 x}{\cos x}$$v' = \frac{\begin{vmatrix} \cos x & 0 \\ -\sin x & \tan x \end{vmatrix}}{W} = \cos x \tan x = \sin x$
5. **积分求 $u(x), v(x)$**：
	- **计算 $u(x)$**：$u'(x) = -\frac{\sin^2 x}{\cos x} = -\frac{1 - \cos^2 x}{\cos x} = -\sec x + \cos x$ 积分得：$u(x) = -\int \sec x \, dx + \int \cos x \, dx = -\ln|\sec x + \tan x| + \sin x + C_1$ （取特解时 $C_1 = 0$，因常数项可并入齐次通解 ）
	    
	- **计算 $v(x)$**：$v'(x) = \sin x$，积分得：$v(x) = -\cos x + C_2$ （同理，取 $C_2 = 0$ ）
### 构造特解 $y_P$

将 $u(x), v(x)$ 代入 $y_P = u(x) \cos x + v(x) \sin x$：$\begin{align*} y_P &= \left(-\ln|\sec x + \tan x| + \sin x\right) \cos x + \left(-\cos x\right) \sin x \\ &= -\cos x \cdot \ln|\sec x + \tan x| + \sin x \cos x - \sin x \cos x \\ &= -\cos x \cdot \ln|\sec x + \tan x| \end{align*}$

### 四、两种方法的对比与选择

|方法|优点|缺点|适用场景|
|---|---|---|---|
|待定系数法|计算简单，形式明确|仅适用于 $f(x)$ 为 “特殊形式”|$f(x)$ 是多项式、指数、三角及乘积|
|常数变易法|通用，对 $f(x)$ 无限制|计算繁琐（需积分、解方程组）|所有非齐次线性 ODE|

### 五、总结：特解求解的完整流程

1. **识别方程类型**：确认是常系数二阶非齐次线性 ODE $a y'' + b y' + c y = f(x)$。
2. **求齐次通解 $y_H$**：用特征方程法找两个线性无关解，构造 $y_H = C_1 y_1 + C_2 y_2$。
3. **选特解方法**：
    - 若 $f(x)$ 是**多项式、指数、三角、或它们的乘积**，用**待定系数法**，按 $f(x)$ 形式假设 $y_P$，代入解系数。
    - 若 $f(x)$ 是**任意形式**（如 $\tan x, \frac{1}{x}$ 等），用**常数变易法**，设 $y_P = u(x) y_1 + v(x) y_2$，解方程组求 $u(x), v(x)$。
4. **构造通解**：$y = y_H + y_P$，代入初始条件确定 $C_1, C_2$（若有）。

## 特解 $y_P$ 的其他求解思路：
### 三、方法 3：叠加原理（拆分非齐次项，化繁为简）

#### 核心思想

若非齐次项 $f(x)$ 可拆分为 **多个简单函数的和**（如 $f(x) = f_1(x) + f_2(x) + \dots + f_n(x)$ ），则根据线性方程的**叠加性**：

> 若 $y_{P1}$ 是 $a y'' + b y' + c y = f_1(x)$ 的特解，$y_{P2}$ 是 $a y'' + b y' + c y = f_2(x)$ 的特解，…，$y_{Pn}$ 是 $a y'' + b y' + c y = f_n(x)$ 的特解，则 $y_P = y_{P1} + y_{P2} + \dots + y_{Pn}$ 是原方程 $a y'' + b y' + c y = f_1(x) + f_2(x) + \dots + f_n(x)$ 的特解。

#### 操作步骤

1. **拆分非齐次项**：将 $f(x)$ 拆成几个 “能用待定系数法解决” 的简单函数（如 $e^x, \sin x, x^2$ 等）。
2. **分别求特解**：对每个拆分后的 $f_i(x)$，用**待定系数法**找对应的特解 $y_{Pi}$。
3. **叠加得总特解**：$y_P = y_{P1} + y_{P2} + \dots + y_{Pn}$。

#### 示例：解方程 $y'' + y = e^x + \sin x$

- **拆分非齐次项**：$f(x) = f_1(x) + f_2(x)$，其中 $f_1(x) = e^x$，$f_2(x) = \sin x$。
    
- **求 $f_1(x) = e^x$ 对应的特解 $y_{P1}$**： 齐次方程 $y'' + y = 0$ 的特征根 $k = \pm i$，$\alpha = 1$ 不是特征根（因 $1^2 + 1 \neq 0$ ）。 设 $y_{P1} = A e^x$，代入方程：$(A e^x)'' + A e^x = e^x$ → $A e^x + A e^x = e^x$ → $2A = 1$ → $A = 1/2$。 故 $y_{P1} = \frac{1}{2} e^x$。
    
- **求 $f_2(x) = \sin x$ 对应的特解 $y_{P2}$**：$\alpha \pm i\beta = 0 \pm i \cdot 1$ 是特征根（因 $k = \pm i$ 是齐次方程的根 ）。 设 $y_{P2} = x (B \cos x + C \sin x)$（单根，乘 x ），代入方程（过程参考前文 “待定系数法 - 情况 3” ），解得 $B = -1/2, C = 0$，故 $y_{P2} = -\frac{1}{2} x \cos x$。
    
- **叠加得总特解**：$y_P = y_{P1} + y_{P2} = \frac{1}{2} e^x - \frac{1}{2} x \cos x$。
#### 适用场景

- 非齐次项 $f(x)$ 是**多个 “简单形式” 的和**（如多项式 + 指数、指数 + 三角函数等 ）。
- 优势：把复杂 $f(x)$ 拆成简单部分，分别用待定系数法解决，避免直接假设复杂的特解形式。

### 四、方法 4：拉普拉斯变换法（适合初值问题，“微分→代数”）

#### 核心思想

利用**拉普拉斯变换**的线性性和微分性质，将**微分方程**转化为**代数方程**，求解代数方程后，再通过**逆拉普拉斯变换**得到特解（或通解）。特别适合**有初始条件**（如 $y(0), y'(0)$ ）的非齐次方程。

#### 拉普拉斯变换的关键性质

1. **线性性**：$\mathcal{L}\{a f(x) + b g(x)\} = a \mathcal{L}\{f(x)\} + b \mathcal{L}\{g(x)\}$
2. **微分性质**：$\mathcal{L}\{y'(x)\} = s Y(s) - y(0)$，$\mathcal{L}\{y''(x)\} = s^2 Y(s) - s y(0) - y'(0)$ （其中 $Y(s) = \mathcal{L}\{y(x)\}$ 是 $y(x)$ 的拉普拉斯变换 ）

#### 操作步骤

1. **对微分方程取拉普拉斯变换**： 原方程 $a y'' + b y' + c y = f(x)$，两边取拉普拉斯变换，利用微分性质转化为关于 $Y(s)$ 的代数方程。
    
2. **代入初始条件，解代数方程**： 若有初始条件 $y(0) = y_0, y'(0) = y_1$，代入后解出 $Y(s)$。
    
3. **求逆拉普拉斯变换**： 通过查表或部分分式分解，求 $Y(s)$ 的逆变换 $\mathcal{L}^{-1}\{Y(s)\}$，得到 $y(x)$（包含齐次解和特解 ）。
#### 示例：用拉普拉斯变换法解 $y'' + y = \sin x$，初始条件 $y(0) = 0, y'(0) = 0$

1. **对原方程取拉普拉斯变换**： 左边：$\mathcal{L}\{y'' + y\} = \mathcal{L}\{y''\} + \mathcal{L}\{y\} = s^2 Y(s) - s y(0) - y'(0) + Y(s)$ 代入初始条件 $y(0) = 0, y'(0) = 0$，得 $(s^2 + 1) Y(s)$。
    
    右边：$\mathcal{L}\{\sin x\} = \frac{1}{s^2 + 1}$（查表得 ）。
    
    因此代数方程：$(s^2 + 1) Y(s) = \frac{1}{s^2 + 1}$，解得 $Y(s) = \frac{1}{(s^2 + 1)^2}$。
    
2. **求逆拉普拉斯变换**： 查表或用部分分式分解，知 $\mathcal{L}^{-1}\left\{ \frac{1}{(s^2 + 1)^2} \right\} = \frac{1}{2} (\sin x - x \cos x)$（推导需用卷积定理，此处略 ）。
    
    因此，方程的解（特解 + 齐次解，因初始条件为 0，齐次解部分为 0 ）：$y(x) = \frac{1}{2} (\sin x - x \cos x)$，即特解 $y_P = \frac{1}{2} (\sin x - x \cos x)$（与待定系数法结果一致 ）。
#### 适用场景

- 非齐次方程**带有初始条件**（初值问题），尤其是 $f(x)$ 形式复杂，用待定系数法或常数变易法麻烦时。
- 优势：将微分方程转化为代数方程，避免 “找特解 + 齐次通解” 的分步操作，直接得到满足初始条件的解。

### 五、四种方法的完整对比（总结表）

|方法|核心逻辑|适用场景|优点|缺点|
|---|---|---|---|---|
|待定系数法|假设特解形式与 $f(x)$ 同类型，代入解系数|$f(x)$ 是多项式、指数、三角及乘积|计算简单，形式明确|仅适用于特殊形式 $f(x)$|
|常数变易法|将齐次通解的常数换为函数，代入非齐次方程解函数|所有非齐次线性 ODE|通用，对 $f(x)$ 无限制|计算繁琐（积分、解方程组）|
|叠加原理|拆分 $f(x)$ 为简单部分，分别求特解再叠加|$f(x)$ 是多个简单函数的和|简化复杂 $f(x)$ 的特解假设|依赖于其他方法（如待定系数法）|
|拉普拉斯变换法|微分方程→代数方程→逆变换，直接求满足初始条件的解|带初始条件的非齐次方程|适合初值问题，避免分步找特解|需记忆拉普拉斯变换表，涉及积分变换|

### 六、方法选择的优先级

1. 优先看 $f(x)$ 形式：
    
    - 若 $f(x)$ 是**多项式、指数、三角及乘积**，用 **待定系数法**（最快最直接 ）。
    - 若 $f(x)$ 是**多个简单形式的和**，用 **叠加原理 + 待定系数法**（拆分后更简单 ）。
2. 若 $f(x)$ 是**任意形式**（如 $\tan x, \frac{1}{x}$ ），用 **常数变易法**（唯一通用方法 ）。
    
3. 若题目**给了初始条件**，且 $f(x)$ 适合拉普拉斯变换，用 **拉普拉斯变换法**（直接出带初始条件的解 ）。

## 关于 A，B 的问题
在二阶常系数线性齐次微分方程的通解 $y_H(x) = A \cdot y_1(x) + B \cdot y_2(x)$ 里，**A 和 B 是任意常数**，作用如下：

#### 1. 体现解的 “任意性”

微分方程的通解要涵盖**所有可能的解**。因为 $y_1(x)$ 和 $y_2(x)$ 是线性无关的 “基础解”，通过给 A、B 赋不同的值（实数或复数，取决于方程场景 ），就能组合出方程的**全部解** 。
比如之前例子里 $y_H(x) = A\cos(x) + B\sin(x)$，当 $A=1,B=0$ 时是 $\cos(x)$；$A=0,B=1$ 时是 $\sin(x)$；$A=2,B=-3$ 时是 $2\cos(x) - 3\sin(x)$ ，这些都是齐次方程的解，靠 $A,B$ 调整覆盖所有可能。

#### 2. 由 “初始条件 / 边界条件” 确定

实际问题中，微分方程会搭配**初始条件**（比如 $y(x_0)=y_0, \, y'(x_0)=y_0'$ ）或**边界条件**（比如 $y(a)=y_a, \, y(b)=y_b$ ）。代入通解后，解关于 A、B 的方程组，就能确定唯一的 A 和 B，得到**特解**（满足具体条件的解 ）。

  

比如方程 $y'' + y = 0$ ，若初始条件是 $y(0)=1, \, y'(0)=0$ ： 通解 $y = A\cos(x) + B\sin(x)$，代入 $x=0$：

  

- $y(0)=A\cos(0) + B\sin(0) = A = 1$
- 求导 $y' = -A\sin(x) + B\cos(x)$，代入 $x=0$ 得 $y'(0)=B = 0$
所以 $A=1, B=0$，特解就是 $y = \cos(x)$ 。

简单说，A 和 B 是 “通解里的自由参数”，让通解能包含方程的所有解；遇到具体问题时，靠额外条件（初始 / 边界条件 ）“锁定” 它们，得到符合实际需求的特解 。

## 上面某个步骤的证明

 1. **定理内容（Theorem 3.35）** 若 $y_1$ 和 $y_2$ 是齐次方程 $$P(x) \cdot \frac{d^2 y}{dx^2} + Q(x) \cdot \frac{dy}{dx} + R(x) \cdot y = 0$$ 的两个解，那么对于任意常数 A 和 B，函数：$$y = A \cdot y_1 + B \cdot y_2$$ 也是该齐次方程的解。
    
2. **证明思路（验证解的性质）** 要证明 $$y = A y_1 + B y_2$$ 是解，只需将 y 代入齐次方程，验证等式成立： **首先求导数**： 一阶导数：$$\frac{dy}{dx} = A \cdot \frac{dy_1}{dx} + B \cdot \frac{dy_2}{dx}$$ 二阶导数：$$\frac{d^2 y}{dx^2} = A \cdot \frac{d^2 y_1}{dx^2} + B \cdot \frac{d^2 y_2}{dx^2}$$
    

**然后将 y、$\frac{dy}{dx}$、$\frac{d^2 y}{dx^2}$ 代入齐次方程左边**：$$\begin{align*} & P(x) \cdot \frac{d^2 y}{dx^2} + Q(x) \cdot \frac{dy}{dx} + R(x) \cdot y \\ =& P(x) \left( A \cdot \frac{d^2 y_1}{dx^2} + B \cdot \frac{d^2 y_2}{dx^2} \right) + Q(x) \left( A \cdot \frac{dy_1}{dx} + B \cdot \frac{dy_2}{dx} \right) + R(x) \left( A \cdot y_1 + B \cdot y_2 \right) \\ =& A \left( P(x) \cdot \frac{d^2 y_1}{dx^2} + Q(x) \cdot \frac{dy_1}{dx} + R(x) \cdot y_1 \right) + B \left( P(x) \cdot \frac{d^2 y_2}{dx^2} + Q(x) \cdot \frac{dy_2}{dx} + R(x) \cdot y_2 \right) \end{align*}$$ 由于 $y_1$ 和 $y_2$ 是齐次方程的解，所以：$$P(x) \cdot \frac{d^2 y_1}{dx^2} + Q(x) \cdot \frac{dy_1}{dx} + R(x) \cdot y_1 = 0$$$$P(x) \cdot \frac{d^2 y_2}{dx^2} + Q(x) \cdot \frac{dy_2}{dx} + R(x) \cdot y_2 = 0$$ 因此，代入后整个式子等于 $$A \cdot 0 + B \cdot 0 = 0$$，满足齐次方程。

## 关于齐次微分方程的更多的一些证明
这是**二阶常系数线性齐次微分方程通解定理（Theorem 3.39）的完整证明**，核心是通过 “假设解的形式 + 代入验证 + 分类讨论”，推导通解的三种情况，分步骤拆解：

### 1. 第一步：假设 “指数函数形式解”，关联辅助方程

要解微分方程 $a \cdot \frac{d^2 y}{dx^2} + b \cdot \frac{dy}{dx} + c \cdot y = 0$，先问：**什么样的 k，能让 $y = e^{kx}$ 成为解？**
对 $y = e^{kx}$ 求导：
- 一阶导 $\frac{dy}{dx} = k e^{kx}$
- 二阶导 $\frac{d^2 y}{dx^2} = k^2 e^{kx}$
把 y、$y'$、$y''$ 代入微分方程：$\begin{align*} a \cdot \frac{d^2 y}{dx^2} + b \cdot \frac{dy}{dx} + c \cdot y &= a \cdot k^2 e^{kx} + b \cdot k e^{kx} + c \cdot e^{kx} \\ &= (a k^2 + b k + c) \cdot e^{kx} \end{align*}$
因为 $e^{kx} \neq 0$（指数函数恒正 ），所以要让上式等于 0，必须满足：$\boldsymbol{a k^2 + b k + c = 0}$
这就是**辅助方程（特征方程）**！
→ 结论：$y = e^{kx}$ 是微分方程的解 **当且仅当** k 是辅助方程的根。

### 2. 第二步：分类讨论辅助方程的根，推导通解

辅助方程是二次方程，根据**判别式 $\Delta = b^2 - 4ac$** 的符号，分 3 种情况
#### Case 1：$\Delta > 0$（两个不同实根 $k_1, k_2$ ）
此时，辅助方程有两个不同实根 $k_1 \neq k_2$，对应微分方程的两个解：$y_1 = e^{k_1 x}, \quad y_2 = e^{k_2 x}$
需要验证这两个解**线性无关**： 因为 $k_1 \neq k_2$，不存在常数 C 使得 $e^{k_1 x} = C \cdot e^{k_2 x}$（否则 $k_1 = k_2$，矛盾 ）。根据线性无关的定义（若 $A y_1 + B y_2 = 0$ 对所有 x 成立，则 $A = B = 0$ ），$y_1, y_2$ 线性无关。
因此，微分方程的通解为：$\boldsymbol{y_H = A e^{k_1 x} + B e^{k_2 x}}$ （$A, B$ 为任意常数，涵盖所有解 ）
#### Case 2：$\Delta = 0$（一个二重实根 k ）

此时，辅助方程有一个二重实根 $k = -\frac{b}{2a}$，对应微分方程的一个解：$y_1 = e^{k x}$

  

但二阶微分方程需要**两个线性无关的解**，所以需要找第二个解。假设第二个解为 $y_2 = x \cdot e^{k x}$，验证是否满足方程：

  

对 $y_2 = x e^{k x}$ 求导：

  

- 一阶导 $y_2' = e^{k x} + k x e^{k x} = (1 + k x) e^{k x}$
- 二阶导 $y_2'' = k e^{k x} + k(1 + k x) e^{k x} = (2k + k^2 x) e^{k x}$

  

代入微分方程左边：$\begin{align*} a y_2'' + b y_2' + c y_2 &= a(2k + k^2 x) e^{k x} + b(1 + k x) e^{k x} + c x e^{k x} \\ &= \left[ (a k^2 + b k + c) x + (2a k + b) \right] e^{k x} \end{align*}$

  

因为 k 是辅助方程的二重根，所以：

  

- $a k^2 + b k + c = 0$（辅助方程本身 ）
- $2a k + b = 0$（二重根的性质，求导后代入辅助方程可得 ）

  

因此，左边等于 $0 \cdot e^{k x} = 0$，即 $y_2 = x e^{k x}$ 也是解。

  

又因为 $y_1 = e^{k x}$ 和 $y_2 = x e^{k x}$ 线性无关（无法表示为彼此的常数倍 ），所以通解为：$\boldsymbol{y_H = (A x + B) e^{k x}}$ （$A, B$ 为任意常数 ）

#### Case 3：$\Delta < 0$（一对共轭复根 $\alpha \pm i\beta$ ）

此时，辅助方程有两个共轭复根 $k = \alpha \pm i\beta$（$\alpha, \beta$ 为实数，$\beta \neq 0$ ），对应微分方程的两个复值解：$y_1 = e^{(\alpha + i\beta) x}, \quad y_2 = e^{(\alpha - i\beta) x}$

  

但实际问题中通常需要**实值解**，利用**欧拉公式 $e^{i\theta} = \cos\theta + i\sin\theta$** ，将复指数解转化为三角函数形式：

  

- $e^{(\alpha + i\beta) x} = e^{\alpha x} \cdot e^{i\beta x} = e^{\alpha x} (\cos\beta x + i\sin\beta x)$
- $e^{(\alpha - i\beta) x} = e^{\alpha x} \cdot e^{-i\beta x} = e^{\alpha x} (\cos\beta x - i\sin\beta x)$

  

将这两个复值解线性组合（取 $C = \frac{1}{2}, D = \frac{1}{2}$ 得实部；取 $C = \frac{1}{2i}, D = -\frac{1}{2i}$ 得虚部 ），可得到两个实值解：$y_1 = e^{\alpha x} \cos\beta x, \quad y_2 = e^{\alpha x} \sin\beta x$

  

验证这两个解线性无关（无法表示为彼此的常数倍 ），因此微分方程的实值通解为：$\boldsymbol{y_H = e^{\alpha x} (A \cos\beta x + B \sin\beta x)}$ （$A, B$ 为任意实常数 ）
