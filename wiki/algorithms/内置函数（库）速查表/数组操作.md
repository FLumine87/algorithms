## 一、原生数组基本操作

| 操作类型          | 语法示例                                     | 说明            |
| ------------- | ---------------------------------------- | ------------- |
| **声明**        | `int arr[5];`                            | 声明大小为5的整型数组   |
| **完全初始化**     | `int arr[5] = {1, 2, 3, 4, 5};`          | 声明时初始化所有元素    |
| **部分初始化**     | `int arr[5] = {1, 2};`                   | 前两个元素初始化，其余为0 |
| **自动推导大小**    | `int arr[] = {1, 2, 3, 4, 5};`           | 编译器自动计算大小为5   |
| **零初始化**      | `int arr[5]{};`                          | 所有元素初始化为0     |
| **元素访问**      | `arr[2] = 10; int x = arr[0];`           | 通过索引访问，索引从0开始 |
| **遍历(传统)**    | `for(int i=0; i<5; ++i) arr[i];`         | 使用索引遍历        |
| **遍历(范围for)** | `for(int val : arr) cout << val;`        | C++11范围for循环  |
| **获取大小**      | `int size = sizeof(arr)/sizeof(arr[0]);` | 计算数组元素个数      |
| **多维声明**      | `int matrix[3][4];`                      | 声明3行4列的二维数组   |
| **多维初始化**     | `int m[2][3] = {{1,2,3},{4,5,6}};`       | 二维数组初始化       |
