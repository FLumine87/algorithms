| 函数                    | 语法                                           | 功能描述   |
| --------------------- | -------------------------------------------- | ------ |
| `accumulate`          | `accumulate(begin, end, init);`              | 累加（求和） |
| `inner_product`       | `inner_product(begin1, end1, begin2, init);` | 内积（点积） |
| `partial_sum`         | `partial_sum(begin, end, dest);`             | 部分和    |
| `adjacent_difference` | `adjacent_difference(begin, end, dest);`     | 相邻差    |
| `iota`                | `iota(begin, end, value);`                   | 用连续值填充 |
