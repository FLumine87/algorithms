### 1.1 创建和初始化

| 操作 | 语法 | 说明 |
|------|------|------|
| **创建空列表** | `lst = []` 或 `lst = list()` | 创建空列表 |
| **创建带元素列表** | `lst = [1, 2, 3, 4, 5]` | 直接初始化 |
| **列表推导式** | `lst = [x for x in range(10)]` | 快速生成列表 |
| **重复列表** | `lst = [0] * 5` | 创建[0, 0, 0, 0, 0] |
| **转换创建** | `lst = list("hello")` | 从字符串创建 |
| **从range创建** | `lst = list(range(5))` | 创建[0, 1, 2, 3, 4] |

### 1.2 访问和修改

| 操作 | 语法 | 说明 |
|------|------|------|
| **索引访问** | `value = lst[2]` | 获取第3个元素（从0开始） |
| **负索引访问** | `value = lst[-1]` | 获取最后一个元素 |
| **切片访问** | `sub = lst[1:4]` | 获取索引1到3的元素 |
| **步长切片** | `sub = lst[::2]` | 每两个取一个 |
| **反转切片** | `rev = lst[::-1]` | 反转列表 |
| **修改元素** | `lst[2] = 10` | 修改单个元素 |
| **切片修改** | `lst[1:3] = [20, 30]` | 修改多个元素 |
| **检查存在** | `if x in lst:` | 检查元素是否在列表中 |

### 1.3 添加和删除

| 操作 | 语法 | 说明 |
|------|------|------|
| **末尾添加** | `lst.append(x)` | 添加单个元素 |
| **任意位置插入** | `lst.insert(i, x)` | 在索引i处插入x |
| **扩展列表** | `lst.extend(iterable)` | 添加多个元素 |
| **合并列表** | `lst1 + lst2` | 返回新列表 |
| **删除末尾** | `x = lst.pop()` | 删除并返回最后一个元素 |
| **删除指定位置** | `x = lst.pop(i)` | 删除并返回索引i的元素 |
| **删除值** | `lst.remove(x)` | 删除第一个值为x的元素 |
| **清空列表** | `lst.clear()` | 清空所有元素 |
| **删除引用** | `del lst[i]` | 删除索引i的元素 |
| **删除切片** | `del lst[2:5]` | 删除切片元素 |

### 1.4 列表方法

| 方法 | 语法 | 说明 |
|------|------|------|
| `append()` | `lst.append(x)` | 添加元素到末尾 |
| `extend()` | `lst.extend(iterable)` | 扩展列表 |
| `insert()` | `lst.insert(i, x)` | 指定位置插入 |
| `remove()` | `lst.remove(x)` | 删除第一个匹配元素 |
| `pop()` | `lst.pop([i])` | 删除并返回元素 |
| `clear()` | `lst.clear()` | 清空列表 |
| `index()` | `lst.index(x[, start[, end]])` | 返回元素索引 |
| `count()` | `lst.count(x)` | 统计元素出现次数 |
| `sort()` | `lst.sort(key=None, reverse=False)` | 原地排序 |
| `reverse()` | `lst.reverse()` | 原地反转 |
| `copy()` | `lst.copy()` | 浅拷贝列表 |

### 1.5 列表操作符

| 操作符 | 语法 | 说明 |
|--------|------|------|
| **拼接** | `lst1 + lst2` | 合并两个列表 |
| **重复** | `lst * 3` | 重复列表3次 |
| **比较** | `lst1 == lst2` | 判断是否相等 |
| **成员测试** | `x in lst` | 判断元素是否存在 |
| **长度** | `len(lst)` | 获取列表长度 |

## 二、Python 内置函数用于列表

| 函数 | 语法 | 说明 |
|------|------|------|
| `len()` | `len(lst)` | 返回列表长度 |
| `max()` | `max(lst)` | 返回最大值 |
| `min()` | `min(lst)` | 返回最小值 |
| `sum()` | `sum(lst)` | 求和（仅数字） |
| `sorted()` | `sorted(lst)` | 返回排序后的新列表 |
| `reversed()` | `reversed(lst)` | 返回反转迭代器 |
| `enumerate()` | `enumerate(lst)` | 返回(索引, 值)对 |
| `zip()` | `zip(lst1, lst2)` | 合并多个列表 |
| `map()` | `map(func, lst)` | 对每个元素应用函数 |
| `filter()` | `filter(func, lst)` | 过滤元素 |
| `any()` | `any(lst)` | 任意元素为True则返回True |
| `all()` | `all(lst)` | 所有元素为True则返回True |
| `list()` | `list(iterable)` | 将可迭代对象转为列表 |
