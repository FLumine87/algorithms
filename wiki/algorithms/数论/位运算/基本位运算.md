## C++ 常见位运算总结

### 1. **基本位运算符**

| 运算符  | 名称   | 示例       | 说明              |
| ---- | ---- | -------- | --------------- |
| `&`  | 按位与  | `a & b`  | 两位都为1则1         |
| `\|` | 按位或  | `a \| b` | 至少一位为1则1        |
| `^`  | 按位异或 | `a ^ b`  | 两位不同则1          |
| `~`  | 按位取反 | `~a`     | 0变1,1变0         |
| `<<` | 左移   | `a << n` | 左移n位，低位补0       |
| `>>` | 右移   | `a >> n` | 右移n位（符号位处理依赖类型） |

### 2. **常用位运算技巧**

#### **判断奇偶**
```cpp
if (n & 1)  // 奇数
if (!(n & 1)) // 偶数
```

#### **乘除2的幂**
```cpp
n << 1;   // n * 2
n >> 1;   // n / 2（整数除法）
n << 3;   // n * 8
```

#### **交换两数（不使用临时变量）**
```cpp
a ^= b;
b ^= a;
a ^= b;
```

#### **获取/设置/清除位**
```cpp
// 获取第k位（0-indexed）
int getBit(int n, int k) {
    return (n >> k) & 1;
}

// 设置第k位为1
int setBit(int n, int k) {
    return n | (1 << k);
}

// 清除第k位（设为0）
int clearBit(int n, int k) {
    return n & ~(1 << k);
}

// 翻转第k位
int toggleBit(int n, int k) {
    return n ^ (1 << k);
}
```

#### **清除最低位的1**
```cpp
n & (n - 1)  // 将最低位的1变成0
// 应用：判断是否为2的幂
bool isPowerOfTwo(int n) {
    return n > 0 && (n & (n - 1)) == 0;
}
```

#### **获取最低位的1**
```cpp
n & -n  // 返回只保留最低位1的值
```

### 3. **实用应用场景**

#### **位掩码（Bitmask）**
```cpp
// 定义标志
const int FLAG_A = 1 << 0;  // 1
const int FLAG_B = 1 << 1;  // 2
const int FLAG_C = 1 << 2;  // 4

int flags = 0;

// 设置标志
flags |= FLAG_A;

// 检查标志
if (flags & FLAG_A) { }

// 清除标志
flags &= ~FLAG_A;

// 切换标志
flags ^= FLAG_A;
```

#### **枚举子集**
```cpp
// 枚举掩码m的所有子集
int m = 0b1101;
for (int sub = m; sub; sub = (sub - 1) & m) {
    // 处理子集sub
}
```

#### **计算二进制中1的个数**
```cpp
int countBits(int n) {
    int count = 0;
    while (n) {
        count++;
        n &= (n - 1);  // 清除最低位1
    }
    return count;
}
// C++20内置：std::popcount(unsigned)
```
