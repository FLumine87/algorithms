## 定义[](https://oi-wiki.org/basic/selection-sort/#%E5%AE%9A%E4%B9%89 "Permanent link")

选择排序（英语：Selection sort）是一种简单直观的排序算法。它的工作原理是每次找出第 𝑖![](data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7 "i") 小的元素（也就是 𝐴𝑖..𝑛![](data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7 "A_{i..n}") 中最小的元素），然后将这个元素与数组第 𝑖![](data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7 "i") 个位置上的元素交换。

## 性质[](https://oi-wiki.org/basic/selection-sort/#%E6%80%A7%E8%B4%A8 "Permanent link")

### 稳定性[](https://oi-wiki.org/basic/selection-sort/#%E7%A8%B3%E5%AE%9A%E6%80%A7 "Permanent link")

选择排序的稳定性取决于其具体实现。

倘若使用链表实现，由于链表的任意位置插入和删除均为 𝑂(1)![](data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7 "O(1)")，故无需使用 swap（交换两个元素）操作：每次从未排序部分选择最小元素（若有多个，选取第 1 个）后，将其插入到未排序部分的第 1 个元素之前，这样就能够保证稳定性。

假如使用数组实现（OI 中一般的实现方式），由于数组任意位置插入和删除均为 𝑂(𝑛)![](data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7 "O(n)")，故只能使用 swap 将未排序部分的元素移到已排序部分。swap 操作使得数组实现的选择排序不稳定。

下面给出的实现示例均是基于数组元素的交换，因此均为 **不稳定的**。

### 时间复杂度[](https://oi-wiki.org/basic/selection-sort/#%E6%97%B6%E9%97%B4%E5%A4%8D%E6%9D%82%E5%BA%A6 "Permanent link")

选择排序的最优时间复杂度、平均时间复杂度和最坏时间复杂度均为 𝑂(𝑛2)![](data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7 "O(n^2)")。

```cpp
#include <utility>

void selection_sort(int* a, int n) {
  for (int i = 1; i < n; ++i) {
    int ith = i;
    for (int j = i + 1; j <= n; ++j) {
      if (a[j] < a[ith]) {
        ith = j;
      }
    }
    std::swap(a[i], a[ith]);
  }
}
```
