### 排序算法空间复杂度总结


| 排序算法 | 空间复杂度 | 说明 |
| :--- | :--- | :--- |
| 插入排序 | O (1) | 原地排序，仅使用常数个额外空间。 |
| 选择排序 | O (1) | 原地排序，仅使用常数个额外空间。 |
| 冒泡排序 | O (1) | 原地排序，仅使用常数个额外空间。 |
| 希尔排序 | O (1) | 原地排序，希尔增量序列的实现通常只使用常数级额外空间。 |
| 归并排序 | O (n) | 需要与待排序数组同等大小的额外空间来进行归并操作。 |
| 堆排序 | O (1) | 原地排序，建堆和调整堆的过程只使用常数级额外空间。 |
| 快速排序 | O (log n) ~ O (n) | 取决于递归深度。平均情况下为 O (log n)，最坏情况下为 O (n)。 |
| 三路快排 | O (log n) ~ O (n) | 与经典快排相同，递归深度决定了空间复杂度。 |
| **桶排序** | **O (n + k)** | 需要额外的桶空间，k 为桶的数量。最坏情况需要 n 个桶。 |
| **基数排序** | **O (n + k)** | 需要额外的桶空间（计数排序作为子程序），k 为基数的范围（通常 k=10）。 |
