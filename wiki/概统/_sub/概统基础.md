### **第一部分：总体与样本**

这是统计学一切推理的起点，区分了我们要研究的对象和我们实际观察到的数据。

1.  **总体**
    *   **定义**： 所有感兴趣的个体、项目或测量值的**完整集合**。总体的大小可以是有限的（如“2023年所有注册选民”），也可以是无限的（如“未来所有可能生产的灯泡”）。
    *   **关键特征**：
        *   **参数**： 描述总体特征的数值度量（如总体均值 `μ`，总体标准差 `σ`）。参数通常是固定但**未知**的。
        *   **研究目标**： 统计分析的终极目标是了解总体。

2.  **样本**
    *   **定义**： 从总体中**选取的一个子集**。它是我们实际收集数据并进行计算的对象。
    *   **关键特征**：
        *   **统计量**： 描述样本特征的数值度量（如样本均值 `x̄`，样本标准差 `s`）。统计量是**已知的**，但会因样本不同而变化。
        *   **作用**： 作为总体的一个“缩影”，用于**推断**总体参数。这种推断总是伴随着不确定性。
    *   **抽样要求**： 样本必须具有**代表性**（能反映总体特征），通常通过**随机抽样**来实现，以避免偏差。

3.  **核心关系**
    *   **从样本到总体**： 描述统计学主要处理样本数据本身。推断统计学则利用样本统计量对总体参数进行估计和假设检验。
    *   **图示**： `总体（参数：μ， σ）` ← **抽样/推断** ← `样本（统计量：x̄， s）`

### **第二部分：描述统计的图表与表格方法**

在对数据进行数值计算前，通过可视化方法初步探索数据分布的形状、中心和离散程度。

1.  **表格方法**
    *   **频率分布表**： 将数据按类别或区间分组，列出每组的频数、相对频率（百分比）和累积频率。
        *   **用途**： 整理无序数据，初步观察数据集中在哪些区间。
    *   **列联表**： 用于展示两个分类变量之间的关系（交叉表）。
        *   **用途**： 观察变量间的关联性。

2.  **图示方法**
    *   **定性数据（分类数据）**：
        *   **条形图**： 用高度或长度表示各类别的频数或比例。类别间是分开的。
        *   **饼图**： 用扇形的角度表示各部分占总体的比例。适合展示构成比。
    *   **定量数据（数值数据）**：
        *   **直方图**： 最常用。将数据范围分成连续的区间，用矩形的高度表示该区间的频数或频率。**用于观察数据分布形状**（对称、偏态、峰态等）。
        *   **茎叶图**： 既能展示分布形状，又能保留原始数据。茎表示高位数字，叶表示低位数字。
        *   **箱线图**： 基于五数概括（见下文）的图形，**用于直观展示数据分布的中心、离散度、偏态和识别异常值**。特别适合多组数据比较。
        *   **散点图**： 用于展示两个定量变量之间的关系（相关性与模式）。

### **第三部分：位置度量**

用于确定数据分布“中心”或“典型值”的统计量。它们描述了数据集中在哪个位置。

1.  **均值**
    *   **计算**： 所有数据之和除以数据个数。`x̄ = (Σx_i) / n`
    *   **特点**：
        *   利用了所有数据信息。
        *   对**极端值（异常值）非常敏感**。
        *   适用于对称分布的数据。

2.  **中位数**
    *   **计算**： 将数据按升序排列后，位于**正中间**的值。
    *   **特点**：
        *   **稳健统计量**，**不受极端值影响**。
        *   适用于偏态分布或有异常值的数据。
        *   是第50个百分位数。

3.  **众数**
    *   **计算**： 数据集中**出现频率最高**的值。
    *   **特点**：
        *   可以有一个、多个或无众数。
        *   适用于所有测量尺度的数据（特别是定性数据）。
        *   对极端值不敏感，但可能不能代表数据的“中心”。

4.  **百分位数**
    *   **定义**： 将数据按大小排序后，使得至少有 `p%` 的数据小于或等于该值，至少有 `(100-p)%` 的数据大于或等于该值的数。中位数是第50百分位数。
    *   **应用**： 标准化考试分数、生长曲线等。

5.  **四分位数**
    *   **定义**： 特殊的百分位数。
        *   **第一四分位数**： 第25百分位数。
        *   **第二四分位数**： 第50百分位数（即中位数）。
        *   **第三四分位数**： 第75百分位数。
    *   **五数概括**： 由**最小值、Q1、中位数、Q3、最大值**组成，是绘制箱线图的基础。

**如何选择？**
*   对称分布： 均值 = 中位数，常用均值。
*   偏态分布或有异常值： 用中位数更具代表性。

### **第四部分：变异程度度量**

仅知道数据中心位置是不够的。变异程度度量描述了数据点围绕中心值的**分散或波动情况**。变异小意味着数据集中，变异大意味着数据分散。

1.  **极差**
    *   **计算**： 最大值 - 最小值。
    *   **特点**： 计算简单，但**只利用了两个极端值，对异常值极其敏感，无法反映中间数据的分布**。

2.  **四分位距**
    *   **计算**： `IQR = Q3 - Q1`。
    *   **特点**：
        *   衡量中间50%数据的离散程度。
        *   **稳健统计量**，**不受极端值影响**。
        *   是识别**异常值**的基础：通常将小于 `Q1 - 1.5*IQR` 或大于 `Q3 + 1.5*IQR` 的值视为异常值。

3.  **方差**
    *   **计算**： 各数据点与均值之差的平方的平均。
        *   **样本方差**： `s² = Σ(x_i - x̄)² / (n-1)` （使用 `n-1` 是为了进行无偏估计）。
        *   **总体方差**： `σ² = Σ(x_i - μ)² / N`。
    *   **特点**：
        *   利用了所有数据信息。
        *   单位是原数据单位的**平方**，解释性不强。

4.  **标准差**
    *   **计算**： 方差的算术平方根。
        *   **样本标准差**： `s = √[Σ(x_i - x̄)² / (n-1)]`
        *   **总体标准差**： `σ = √[Σ(x_i - μ)² / N]`
    *   **特点**：
        *   **最重要、最常用的变异程度度量**。
        *   单位与原数据一致，易于解释。
        *   结合均值使用，**经验法则**表明：对于对称的钟形分布，约68%的数据落在 `均值±1个标准差` 内，约95%落在 `均值±2个标准差` 内，约99.7%落在 `均值±3个标准差` 内。

5.  **变异系数**
    *   **计算**： `CV = (s / x̄) * 100%`
    *   **特点**： 一个**无量纲**的相对离散度指标。**用于比较不同单位或均值差异很大的数据集的变异程度**。

**如何选择？**
*   与中位数配合： 使用 **四分位距**。
*   与均值配合： 使用 **标准差/方差**。
*   比较不同数据集： 使用 **变异系数**。

### **总结与联系**

这四个知识点构成了描述统计学的完整工作流：

1.  **明确对象**： 首先界定你处理的是**总体**还是**样本**，这决定了你后续计算的是参数还是统计量，以及结论的适用范围。
2.  **直观探索**： 通过**图表和表格**对数据进行可视化，初步了解数据的分布形状、模式和潜在问题（如异常值、多峰）。
3.  **量化中心**： 计算**位置度量**（如均值、中位数），用一个数值来概括数据的典型水平。
4.  **量化波动**： 计算**变异程度度量**（如标准差、IQR），用一个数值来概括数据的分散程度，评估位置度量的代表性。
